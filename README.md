# AD-Transformer

这是一个基于PyTorch的Vision Transformer（ViT）的AD诊断与预测模型。

## 🔨 环境依赖

```bash
pip install torch torchvision tqdm matplotlib numpy
```

或使用conda：
```bash
conda install pytorch torchvision tqdm matplotlib numpy -c pytorch
```
